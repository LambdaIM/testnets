## Full nodes and general participants

## Genesis File

Download the genesis file

Verify the SHA256 checksum using:

```bash
sha256sum genesis.json
# a0329465a4e01d5f813a9fe3ab78835fe2d3c91f3b7b77e79405df70eaabab00  genesis.json
```
